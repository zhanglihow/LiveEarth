# LiveEarth!

## Introduction
- 实时动态更新地球壁纸！
- 地球照片抓自日本[himawari-8](http://himawari8.nict.go.jp/)气象卫星官网
- 默认每一小时抓取一次
- 不到50行的python代码
- 感谢 [bitdust](https://github.com/bitdust) 提供的idea
